## **Описание:**
Тестовая реализация мини игры по взлому замков из The Elder Scrols 4 Oblivion.

## **Игровой процесс:**
Стрелками право/лево выбирается бегунок для попытки фиксации.
Пробел/Enter запускает движение бегунка. Повторное нажатие фиксирует бегунок, если его нижний край был около красной полосы, или возвращает его и все зафиксированный бегунки в изначальное положение в случае неудачи.

## **Цель игры:**
Зафиксировать все пять бегунков.
